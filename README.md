# 🐷 PiggyVentura

Juego 2D en desarrollo en Unity como proyecto de práctica.

## 🎮 Descripción
PiggyVentura es un juego de plataformas donde controlas a un cerdito que debe recolectar frutas, evitar trampas y sobrevivir a los peligros del entorno.

## ⚙️ Tecnologías
- Unity 6
- C#
- Git & GitHub

## 🚀 Características
- Movimiento y salto del personaje
- Sistema de recolección de frutas
- Trampas con daño (puas)
- Barriles interactivos con animación
